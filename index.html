Vamos aprender AngularJS?

Aprender o que??

AngularJS, um framework MVC pra desenvolvimento de front-end. É massa, foi o Google que fez.

Quais as vantagens dele?

  • Acaba com aquela mistureba de funcionalidade + apresentação implementado tudo junto
  • É mais rápido fazer telas "ricas" usando Angular do que JQuery
  • Acaba com a bronquidão de ficar manipulando o dom
  • Implementação componentizável. Dá pra fazer uma arquitetura decente, mesmo se o front-end for complexo.
  • O Google que fez --> Ou seja, ele é mantido por gente competente
  • Muita gente já tá usando --> É fácil achar suporte da comunidade
  • E por aí vai
